Om Nom Run

Experience the thrill of endless running with Om Nom as you dash through colorful worlds, collecting candies and avoiding obstacles!

Game Features

  • Endless runner gameplay with increasing difficulty
  • Multiple power-ups and character transformations
  • Beautiful graphics and smooth animations
  • Cross-platform compatibility

How to Play

  • Use arrow keys or swipe to control Om Nom
  • Jump over obstacles and collect candies
  • Unlock special abilities and power-ups
  • Compete for high scores

About Om Nom Run

Om Nom Run is an exciting endless runner game that brings the beloved character from Cut the Rope into a new adventure. As Om Nom races through vibrant environments, players must help him collect candies while avoiding various obstacles.

The game features stunning graphics, intuitive controls, and increasingly challenging gameplay that will keep you coming back for more. With regular updates and new content, Om Nom Run offers endless entertainment for players of all ages.
