About Om Nom Run

Om Nom Run is an exciting endless runner game developed by Famobi. The game features the beloved character Om Nom from the popular Cut the Rope series. Players control Om Nom as he runs through various environments, avoiding obstacles and collecting coins.

The game is built using HTML5 technology, ensuring smooth gameplay across all devices without the need for downloads or installations. With its intuitive controls and engaging gameplay mechanics, Om Nom Run offers hours of entertainment for players of all ages.
